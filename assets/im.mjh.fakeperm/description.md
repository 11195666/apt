# 虚拟权限 · Fake Permission

针对任意 App 单独配置虚拟权限，防止隐私被过度索取。

### 支持的伪装功能

**基础权限伪装：** 通知、推送、相册、通讯录、日历、蓝牙、剪贴板、传感器（防摇一摇）

**定位伪装：** 地图选点 / 地址搜索 / 海拔高度 / 历史记录  

**相机伪装：** 强制相册选择器 / 虚拟拍照替换 / 相册入口按钮

**麦克风伪装：** 完整的麦克风拦截，防止「麦克风被占用」提示

**WiFi 模拟：** 自定义 SSID / BSSID / 历史记录

**机型伪装：** 内置热门机型列表 + 自定义输入

**iOS 版本伪装：** 内置 122 条真实版本数据库 (iOS 15-26)，关联 Build 号

**地区伪装：** 内置 28 个国家/地区选择器，选择后自动填入运营商 MCC/MNC

**语言伪装：** 内置 25 种语言选择器

**UUID 标识符伪装：** IDFV + IDFA 统一为一个 UUID 字段，支持随机生成

**App 版本伪装：** 可自定义 版本号 / 内部号

**屏蔽检测：** 代理/VPN 检测、截图检测、录屏检测

**其他：** 配置导入导出 / App 暂停恢复 / 版本更新检测 / 中英文界面

### Fake Permission · FakeTweak

Individually configure spoofed permissions for any app to prevent excessive privacy requests.

**Permission spoofing:** notifications, push, photos, contacts, calendar, bluetooth, clipboard, sensor (shake-ad blocking)

**Location spoofing:** map picker / address search / altitude / history, covering Amap, Baidu, Tencent LBS SDKs

**Camera spoofing:** force photo library picker / fake capture replacement / album entry button

**Microphone spoofing:** full interception to prevent "mic occupied" errors

**WiFi spoofing:** custom SSID / BSSID / history

**Device model spoofing:** built-in popular models + custom identifier input

**iOS version spoofing:** built-in database of 122 real versions (iOS 15-26) with search and one-tap auto-fill

**Region spoofing:** built-in picker with 28 countries, auto-fills carrier MCC/MNC

**Language spoofing:** built-in picker with 25 languages

**UUID identifier spoofing:** unified IDFV/IDFA field with random generation

**App version spoofing:** customizable CFBundleShortVersionString / CFBundleVersion

**Detection blocking:** proxy/VPN detection, screenshot detection, screen recording detection

**More:** config import/export, app pause/resume, version update checker, Chinese/English UI
