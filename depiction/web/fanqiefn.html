番茄净化

番茄小说(com.dragon.read) 伪装通知和小组件权限。

功能

  • 去除添加小组件的弹窗提醒
  • 去除开启通知提醒
  • 安装即默认生效，无开关

使用说明

如需关闭推送权限弹窗：前往系统 设置 → 番茄小说 → 关闭通知权限

源代码

  • iOS 15.0+
  • 代码在 github 开源，地址：https://github.com/11195666/fanqiefn
