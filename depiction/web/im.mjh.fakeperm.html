虚拟权限 · Fake Permission

针对任意 App 单独配置虚拟权限，防止隐私被过度索取。

支持的权限

  • 通知 / Notification
  • 推送 / Push
  • 定位 — 地图选点 + 地址搜索 + 海拔伪装
  • 相册 / Photos
  • 通讯录 / Contacts
  • 相机 — 强制相册 / 虚拟拍照
  • 麦克风 / Microphone
  • 蓝牙 / Bluetooth
  • 剪贴板 / Clipboard
  • 机型伪装 / Device Model Spoofing

特点

  • 按 App 单独配置，灵活控制
  • 全局开关即时生效，无需注销
  • 中英文界面
  • 兼容 iOS 15.0+
